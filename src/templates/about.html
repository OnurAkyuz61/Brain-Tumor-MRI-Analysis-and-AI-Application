About This Project

Understanding brain tumor detection through MRI analysis

Project Overview

This application uses deep learning to analyze brain MRI scans and detect the presence and type of brain tumors. The model has been trained on the Brain Tumor MRI Dataset from Kaggle, which contains 7023 images of human brain MRI scans classified into four categories:

Glioma

Gliomas are tumors that develop from glial cells, which support and nourish neurons in the brain. They are the most common type of primary brain tumor.

Meningioma

Meningiomas arise from the meninges, the membranes that surround the brain and spinal cord. They are typically slow-growing and often benign.

No Tumor

MRI scans showing normal brain tissue without any tumor present.

Pituitary Tumor

Pituitary tumors develop in the pituitary gland located at the base of the brain. They can affect hormone production and cause various symptoms.

Technology

This application uses the following technologies:

  • TensorFlow/Keras: For building and training the deep learning model
  • Transfer Learning: Using pre-trained MobileNetV2 as the base model
  • Flask: Python web framework for the backend
  • Bootstrap: For responsive and modern UI design
  • Chart.js: For visualizing prediction probabilities

Model Architecture

The model uses transfer learning with MobileNetV2 as the base architecture, which has been pre-trained on the ImageNet dataset. Additional layers have been added on top of the base model for the specific task of brain tumor classification.

Training Process

The model was trained on the Brain Tumor MRI Dataset with data augmentation techniques to improve generalization. The training process included:

  • Image preprocessing and normalization
  • Data augmentation (rotation, zoom, flip, etc.)
  • Transfer learning from a pre-trained model
  • Fine-tuning for the specific classification task
